## git commit 规范

记完一次就提交，commit message 为有改动的笔记本名称

## 排版格式

基本遵循 [中文文案排版指北](https://github.com/sparanoid/chinese-copywriting-guidelines/blob/master/README.zh-Hans.md)

### 例外

- 不使用全角的（），而使用半角的 ()，并且前后空一格
- 不使用「」，使用“”，并且前后空一格
- 句末不加句号

### 额外的

- 不使用行尾空两格的形式进行换行，而是额外一个空行

  - ```markdown
    P1

    P2
    ```

- 行内数学公式前后空一格
  - `句子 $math$ 句子`
- 行间数学公式使用以下形式

  - ```markdown
    $$
    math
    $$
    ```
